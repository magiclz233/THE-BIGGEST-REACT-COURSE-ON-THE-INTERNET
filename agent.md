# 项目速览

- 本仓库收录了 YouTube 播放列表“The Complete React Ecosystem”的随堂素材与练习，是一套从入门到进阶的 React 全路径课程。
- 课程内容按模块编号分布在 `01.` ~ `22.` 目录下，涵盖基础、Hooks、TypeScript、状态管理、动画、测试、Storybook、设计系统、鉴权乃至 React Native 等主题。
- 典型结构：每个小节通常包含 `exercise/README.md`（练习说明）、`exercise/solution`（参考解法）以及 `live coding`（课堂演示代码或起步模版）。部分练习附有配图或 UI 参考。
- 根目录的 `thumb.png` 提供课程封面缩略图。
- 学习建议：按编号顺序结合视频学习，先阅读对应练习说明，尝试独立完成，再对照 `solution` 目录巩固；如需要运行代码，进入对应小节目录后自行安装依赖并启动（通常为 React/Vite 或 CRA 的常规流程）。
